## Phase 1: 프로젝트 초기 세팅 (1주)

### 프롬프트 1-1: 프로젝트 구조 생성
```
Electron + React + TypeScript 프로젝트를 생성해줘.

요구사항:
1. 프로젝트명: work-timer
2. Electron + React 18 + TypeScript 조합
3. Vite 번들러 사용
4. 다음 디렉토리 구조로 생성:
   - src/main/ (Main Process)
   - src/renderer/overlay/ (오버레이 윈도우)
   - src/renderer/dashboard/ (대시보드 윈도우)
   - src/preload/ (Preload Scripts)
   - resources/ (아이콘, 이미지)

5. 필요한 패키지 설치:
   - electron
   - react, react-dom
   - electron-store
   - active-win
   - zustand
   - tailwindcss
   - recharts
   
6. package.json에 다음 스크립트 추가:
   - dev: 개발 모드 실행
   - build: 프로덕션 빌드
   - build:win: Windows 빌드
   - build:mac: macOS 빌드

7. TypeScript 설정 완료
8. Tailwind CSS 설정 완료
9. ESLint + Prettier 설정

프로젝트가 정상적으로 실행되는지 확인까지 해줘.
```

```

요구사항:
```

```
3. idle 시간 계산 함수

   - restingThreshold: 기본 5분

   - 'idle-resting': 5분 idle
   - 'idle-sleeping': 30분 idle

main/index.ts에 통합하고 테스트해줘.
```

### 프롬프트 2-3: 상태 관리 시스템
```
src/main/monitor/state-manager.ts 파일을 생성하고 작업 상태 관리를 구현해줘:

요구사항:
1. WorkState enum 정의:
   - WORKING: 작업중
   - DISTRACTED: 딴짓중
   - RESTING: 쉬는 중
   - EATING: 밥먹는 중
   - SLEEPING: 자는 중

2. StateManager 클래스 구현:
   - currentState: 현재 상태
   - currentProject: 현재 프로젝트 ID
   - sessionStartTime: 세션 시작 시간
   - previousState: 이전 상태 (밥먹는 중에서 복귀용)

3. 메서드:
   - setState(state, projectId?): 상태 변경
   - onProgramChange(programName): 프로그램 변경 처리
   - onIdleDetected(type): idle 감지 처리
   - toggleEating(): 밥먹는 중 토글
   - getCurrentState(): 현재 상태 정보 반환

4. 상태 변경 시 로그 출력
5. 상태 변경 시 'state-changed' 이벤트 발생

process-monitor, idle-detector와 통합해서 자동으로 상태가 전환되는지 테스트해줘.
```
